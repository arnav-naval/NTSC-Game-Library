NTSC Game Library
Authors:

Arnav Naval, Connor Behrend, Niril Jestus, Riley Mann

Description
The NTSC Game Library is a high-performance C-based framework designed for developing classic video games on STM32 microcontrollers. It provides real-time video rendering, sprite manipulation, and sound generation, enabling seamless game development for analog NTSC television displays. The library efficiently handles NTSC signal emulation via SPI, producing precise composite video signals for smooth and colorful graphic

Purpose
The NTSC Game Library enables developers to build retro-style video games on embedded systems while ensuring high-performance video rendering, audio processing, and persistent storage. By leveraging low-level hardware control, the library maximizes efficiency and precision, making it an ideal choice for game development on microcontroller-based platforms.
